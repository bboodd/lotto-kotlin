# 🎰 로또 프로그램 만들기
> 전략 패턴과 파일 기반 이력 관리를 적용한 순수 코틀린 로또 시뮬레이터입니다.

---

## 🚀 주요 기능
- **구매 전략 패턴으로 다양한 구매 방식 적용**
- **파일 기반 이력 관리로 로또 구매 이력 저장**
- **입력값과 로또 규칙 예외처리**

## 🛠 실행 환경
- **Language**: Kotlin 2.2.20+
- **Platform**: JVM
- **Formatting**: Ktlint 준수

## 🏗 주요 설계
### 1. 전략 패턴
- `TicketIssueStrategy` 인터페이스를 통해 자동/수동/반자동 발급 로직을 동적으로 교체할 수 있도록 설계

### 2. 일급 컬렉션
- `LottoTickets` 클래스를 도입하여 티켓 리스트에 대한 검증과 당첨 확인 로직을 캡슐화

## 📊 당첨 규칙 및 상금

| 등수 | 조건 | 상금 |
| :-- | :--- | :--- |
| **1등** | 6개 번호 일치 | 2,100,000,000원 |
| **2등** | 5개 번호 + 보너스 일치 | 60,000,000원 |
| **3등** | 5개 번호 일치 | 1,500,000원 |
| **4등** | 4개 번호 일치 | 50,000원 |
| **5등** | 3개 번호 일치 | 5,000원 |

## 🚀 실행방법
#### 1. `src/main/kotlin/Main.kt`를 실행하여 로또 프로그램을 시작합니다.
#### 2. 콘솔창의 안내에 따라 입금, 구입 매수, 발급 방식(자동/수동/반자동)을 선택하세요.
#### 3. 모든 구매가 완료되면 당첨 번호 추첨을 선택하세요.
